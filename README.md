# VK Quiz App

VK Quiz App — это MVP веб-приложения для проведения интерактивных квизов в реальном времени.

Проект позволяет организатору создавать квизы, запускать комнату, подключать участников по коду, показывать вопросы в реальном времени, принимать ответы, считать баллы и отображать итоговый лидерборд.

## Функциональность

- регистрация и авторизация пользователей;
- личный кабинет организатора;
- создание и настройка квиза;
- добавление вопросов с одиночным и множественным выбором;
- добавление изображения к вопросу через ссылку;
- запуск комнаты квиза;
- подключение участников по коду комнаты;
- показ вопросов в реальном времени через WebSocket;
- таймер для ответа на вопрос;
- подсчёт баллов;
- отображение итогового лидерборда;
- история проведённых квизов;
- страница результатов.

## Как запустить проект

1. Установить Node.js.

2. Клонировать репозиторий:

git clone https://github.com/miracleprime/vk-quiz-app.git

3. Перейти в папку проекта:

cd vk-quiz-app

4. Установить зависимости:

npm install

5. Запустить проект:

npm run dev

6. Открыть в браузере:

http://localhost:3000

## Основной сценарий работы

1. Организатор регистрируется и входит в личный кабинет.
2. Создаёт квиз и добавляет вопросы.
3. Запускает комнату квиза.
4. Участники подключаются по коду комнаты.
5. Организатор показывает вопросы.
6. Участники отвечают в ограниченное время.
7. Система считает баллы.
8. После завершения отображается лидерборд и страница результатов.

## База данных

В проекте используется SQLite.

Основные таблицы:

- `users` — пользователи;
- `quizzes` — квизы;
- `questions` — вопросы;
- `answer_options` — варианты ответов;
- `rooms` — комнаты;
- `participants` — участники;
- `answers` — ответы участников.

## Статус проекта

Проект является MVP-прототипом и реализует основной функционал веб-приложения для проведения интерактивных квизов в реальном времени.

Деплой приложения не выполнялся, так как ссылка на работоспособный продукт является опциональной. Проверка MVP осуществляется локально после установки зависимостей и запуска сервера командой `npm run dev`.
